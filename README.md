# Telco-Customer-Churn-Analysis
Customer Churn Analysis using Python, Pandas, Matplotlib, and Seaborn with EDA and business insights.

## Problem Statement
Customer churn is one of the biggest challenges faced by telecom companies. 
The objective of this project was to analyze customer data, identify factors contributing to churn, and generate actionable insights to improve customer retention.

---

## Dataset Overview
The dataset contains:
- Customer demographic details
- Subscription information
- Payment methods
- Monthly charges
- Tenure
- Internet services
- Churn status

---

- Python

---

## Tasks Performed
- Data Cleaning & Preprocessing
- Handling Missing Values
- Data Type Conversion
- Exploratory Data Analysis (EDA)
- Percentage-based Analysis
- Customer Segmentation
- Data Visualization
- Churn Pattern Identification

---

## Key Insights
- Around 26–27% customers churned.
- Month-to-Month contract customers showed highest churn.
- Senior Citizens had comparatively higher churn rates.
- Higher monthly charges increased churn probability.
- Customers with lower tenure churned more frequently.
- Electronic Check users showed higher churn percentages.

---

## Business Impact
This analysis helps businesses identify high-risk customers and improve customer retention strategies.
